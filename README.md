# Catalyst Analytics

**Author:** Sheravganzoda Amirjon

## 📌 About the Project

Catalyst Analytics is a modern multi-page responsive website created as a frontend portfolio project. The website represents a technology company specializing in Artificial Intelligence, Data Analytics, Machine Learning, and Data Engineering services.

The project was built using **HTML5**, **Tailwind CSS**, and follows responsive design principles for desktop, tablet, and mobile devices.

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/catalyst-analytics.git
```

Open the project folder:

```bash
cd catalyst-analytics
```

Run Tailwind CSS:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
```

Open `index.html` in your browser.
